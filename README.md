
Cirru Parser in Racket
---

Converted from CoffeeScript.

### Usage

```text
raco pkg install git://github.com/Cirru/parser-rkt
```

```rkt
#lang racket
(require cirruparser)

(cirruparser-parse "code" "file")
(cirruparser-pare "code" "file")
```

### Development

Run this as a test:

```text
racket test.rkt
```
